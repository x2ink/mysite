# 笔墨AI

## 作品介绍

笔墨AI是一个集成了多种AI玩法的工具网站，涵盖了AI文生图、AI图生图、AI换脸等多种功能，为用户提供了一个全方位的AI创作平台。

## 项目技术栈

### 框架

- **Vue3**：用于构建用户界面的渐进式框架。
- **Axios**：用于处理HTTP请求的库。
- **Vue-Router**：用于实现单页应用的路由管理。
- **Element-plus**：基于Vue3的UI组件库。

### 前端技术

- **HTML**：用于构建网页结构。
- **CSS**：用于网页样式设计。
- **JavaScript**：用于实现网页交互逻辑。
